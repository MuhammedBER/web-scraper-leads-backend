## 📌 Overview

The backend REST API for the Web Scraper Leads platform. Built with **Spring Boot** and **MySQL**, it scrapes target websites and extracts valuable contact information including emails, phone numbers and social media accounts — all exportable as a CSV file.

---

## ✨ Features

- 🌐 **Web Scraping** — Scrape any list of websites automatically
- 📧 **Email Extraction** — Extract all email addresses from target websites
- 📞 **Phone Number Extraction** — Detect and collect phone numbers
- 📱 **Social Media Extraction** — Extract social media account links
- 📥 **CSV Export** — Download all scraped leads as a CSV file
- 🗄️ **Data Persistence** — Save and manage scraped results in MySQL

## 🚀 Getting Started

### Prerequisites
- Java >= 17
- Maven
- MySQL

### Setup

```bash
# Clone the repo
git clone https://github.com/MuhammedBER/web-scraper-leads-backend.git
cd web-scraper-leads-backend

# Configure your database in application.properties
# Run the app
mvn spring-boot:run
```

### API Docs
Once running, visit:
```
http://localhost:8080/swagger-ui.html
```
